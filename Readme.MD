## المميزات

- تصنيف بفئات متعددة (مرضى، أرامل، أيتام...)
- بحث سريع + عرض جدول وبطاقات

- رفع إثباتات التبرع ومراجعتها
- تصدير CSV
- واجهة عربية RTL كاملة
